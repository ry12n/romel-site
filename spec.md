# Spec — ROMEL Landing Page

**Stack:** Plain HTML5 + CSS3, single `index.html`, zero build step.
**Deploy:** GitHub Pages from `main` root.
**Files:**
- `index.html` — complete page with inline CSS
- `design/design.md` — design record

**Acceptance criteria:**
- Loads over HTTPS with HTTP 200.
- Zero rounded corners, zero gradients, zero warm colours.
- Exactly one primary CTA per view, phrased as access.
- Agent roster displays all 10 names with mono IDs and role labels.
- Form provides simulated success feedback.
